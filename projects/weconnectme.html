WeConnectMe

WeConnectMe is a startup looking to close the gap between remote work and the in-person office experience. With remote work skyrocketing, the workplace needed a professional virtual office to bring back spontaneity. WCM wanted a professional web app design for any type of work culture that can be used by internal employees and prospecting clients. Features include single/group video calls, single/group chats, webinars, virtual events & conferences, and more!

I collaborated with a lead UI Designer, Environment Illustrator, App Developers. Our app developers created initial wireframes that were viable for Phase 1. I created an in-depth design system using Figma’s components. Then the lead designer and I designed the initial UI Design based on the wireframes provided.

Figma Design System

I created a component design system that includes color, type, buttons, and larger components. Using Figma's variation properties, I could create components for all states and situations.
